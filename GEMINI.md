# LT4: WoW Quality-of-Life Suite

**Project Scope:** This addon is designed exclusively for the most recent expansion of World of Warcraft (currently: **Midnight**). It does not maintain backward compatibility for Classic or older Retail versions.

## Core Mandates
- **Performance:** Throttled updates (0.05s) for UI elements.
- **Modularity:** Every feature must be a module.
- **Framework:** Ace3-based.
- **Skinning:** Standardized ElvUI skinning patterns.
